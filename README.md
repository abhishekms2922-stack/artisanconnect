# ArtisanConnect

## Problem
Local Mysore silk and handloom artisans lack digital marketing and direct customer access.

## Solution
ArtisanConnect is a web platform connecting artisans directly with customers.

## Features
- Artisan Login
- Add Products
- View Products
- Order Products
- Firebase Database
- Order Management

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase

## How to Run
1. Open index.html
2. Login as artisan
3. Add products
4. Order products
